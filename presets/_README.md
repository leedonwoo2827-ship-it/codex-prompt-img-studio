# presets 폴더 — 기본 프리셋 강제 추가하는 법

이 폴더에 하위 폴더를 만들면 **프롬프트 빌더의 「프리셋」 탭에 자동으로 나타납니다.**
완료된 카드뉴스 중 마음에 드는 걸 기본 프리셋으로 박아둘 때 씁니다.

## 폴더 구조
```
presets/
  내_카드뉴스_A/
    cover.png        # (선택) 썸네일·예시 이미지 — cover.png/jpg/jpeg/webp
    preset.md        # (선택) 설명·적용 키워드·프롬프트 골격
    meta.json        # (선택) {"description":"...", "keywords":["..."], "builtin": true}
```

- `meta.json` 의 `builtin: true` → **내장 프리셋**(UI에서 삭제 불가, 항상 노출).
- `meta.json` 이 없으면 기본적으로 내장으로 간주합니다.
- UI의 「새 프리셋 저장」으로 만든 것은 `builtin: false` 로 저장되어 삭제할 수 있습니다.

## 추가 방법 (강제 시드)
1. 이 폴더 안에 새 폴더를 만든다 (폴더명이 프리셋 이름).
2. 예시 이미지를 `cover.png` 로 넣는다.
3. (선택) `preset.md` 에 톤/색/구도/키워드를 적는다.
4. 빌더를 새로고침하면 「프리셋」 탭과 빌더 사이드 패널에 나타난다.
